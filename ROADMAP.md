# Roadmap

## Phase 1 — Scanner MVP (current)
- Platform API clients (Polymarket, Kalshi, Manifold, Metaculus)
- Cross-platform event matching
- Spread calculator with fee adjustment
- Discord alerts
- CLI runner

## Phase 2 — Execution Engine
- Polymarket trading client (USDC wallet, order placement)
- Kalshi trading client (authenticated API)
- Position tracker + P&L
- Settlement reconciliation
- Risk limits + position sizing

## Phase 3 — Intelligence Layer
- Multi-platform consensus "fair value" model
- Single-platform mispricing detection (lagging on news)
- Automated execution when spread + confidence exceed threshold
- Kelly criterion position sizing
- Event resolution probability model
